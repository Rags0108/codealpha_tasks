# 🎯 Completed Tasks

## 🌐 Task 1: AI-Powered Language Translation Tool

### Description

An AI-powered language translation application capable of translating text between 50+ languages using automatic language detection and real-time translation.

### Key Features

* Translate text into 50+ languages
* Automatic language detection
* Language swapping functionality
* Copy translated text
* User-friendly Gradio interface
* Real-time translation

### Technologies

* Python
* Gradio
* Deep Translator
* Google Translator API

## 🏥 Task 2: Healthcare FAQ Chatbot

### Description

### Key Features

* NLP Text Preprocessing
* TF-IDF Vectorization
* Cosine Similarity Matching
* Streamlit Web Interface

## 🎯 Task 4: AI-Powered Object Detection and Tracking

### Description

A real-time object detection and multi-object tracking system using YOLO and ByteTrack capable of detecting, tracking, and counting objects in videos or webcam streams.

### Key Features

* Real-Time Object Detection
* Multi-Object Tracking
* Object Counting
* Unique Tracking IDs
* Confidence Score Display
* Output Video Generation

### Technologies

* Python
* OpenCV
* Ultralytics YOLO
* ByteTrack
* NumPy

```text
Task4_Object_Detection/
```
