# 🌐 AI-Powered Language Translation Tool

## 📌 Project Overview

The AI-Powered Language Translation Tool is a web-based application developed as part of the **CodeAlpha AI Internship Program**.

This application enables users to translate text between 50+ languages using automatic language detection and real-time translation. The tool provides a simple and user-friendly interface built with Gradio and Python.

---

## 🚀 Features

* 🌍 Translate text into 50+ languages
* 🔍 Automatic source language detection
* 🔄 Swap source and target languages
* 🧹 Clear input and output text
* 📋 Copy translated text with one click
* 📝 Example translation inputs
* 🎨 Modern and responsive Gradio interface
* ⚡ Fast translation using Google Translator API

---

## 🛠️ Technologies Used

* Python 3.x
* Gradio
* Deep Translator
* Google Translator API

## 📸 Application Workflow

1. Enter text in the Input Text box.
2. Select Source Language.
3. Select Target Language.
4. Click Translate.
5. View the translated output.
6. Use Swap Language to reverse translation.
7. Use Clear Text to reset the fields.
